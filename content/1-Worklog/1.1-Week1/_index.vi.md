### Kết quả đạt được tuần 1:

* **Tạo thành công tài khoản AWS Free Tier 2025** và nắm vững các tính năng miễn phí.

* **Thiết lập quản lý chi phí toàn diện:**
  * Cấu hình billing alerts để giám sát ngân sách
  * Thiết lập budgets để ngăn chặn phí phát sinh bất ngờ
  * Nắm vững giới hạn AWS Free Tier và các thực hành tốt nhất

* **Thành thạo điều hướng AWS Console:**
  * Khám phá các nhóm dịch vụ chính (Compute, Storage, Networking, Database)
  * Hiểu về hạ tầng toàn cầu AWS (Regions, Availability Zones)
  * Học cách tìm kiếm và truy cập dịch vụ hiệu quả

* **Triển khai thành công EC2 instance đầu tiên:**
  * Khởi tạo t2.micro instance trong giới hạn Free Tier
  * Cấu hình security groups cho web traffic (HTTP/HTTPS)
  * Thiết lập kết nối SSH sử dụng key pairs

* **Hoàn thành dự án thực hành deploy web server:**
  * Cài đặt và cấu hình Apache web server
  * Tạo và triển khai trang web HTML cơ bản
  * Xác minh khả năng truy cập công khai và chức năng
  * Tích lũy kinh nghiệm thực tế về quản trị Linux server

* **Phát triển kỹ năng AWS cơ bản** bao gồm quản lý tài khoản, các khái niệm networking cơ bản, và cấp phát tài nguyên cloud.
