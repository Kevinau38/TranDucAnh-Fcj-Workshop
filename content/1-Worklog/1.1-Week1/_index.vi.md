### Kết quả đạt được tuần 1:

* **Kết nối thành công với cộng đồng First Cloud Journey** và hoàn thành quy trình hướng dẫn thực tập.

* **Tạo thành công tài khoản AWS Free Tier 2025** và hoàn thành 5 nhiệm vụ kiếm credit để tích lũy kinh nghiệm thực hành.

* **Thiết lập quản lý chi phí toàn diện:**
  * Cấu hình billing alerts để giám sát ngân sách.
  * Thiết lập budgets để ngăn chặn phí phát sinh bất ngờ.
  * Nắm vững giới hạn AWS Free Tier và các thực hành tốt nhất.

* **Hiểu cách điều hướng AWS Console:**
  * Khám phá các nhóm dịch vụ chính (Compute, Storage, Networking, Database).
  * Hiểu về hạ tầng toàn cầu AWS (Regions, Availability Zones).
  * Học cách tìm kiếm và truy cập dịch vụ hiệu quả.

* **Triển khai thành công EC2 instance đầu tiên:**
  * Khởi tạo t2.micro instance trong giới hạn Free Tier 2025.
  * Cấu hình security groups cho web traffic (HTTP/HTTPS).
  * Thiết lập kết nối SSH sử dụng key pairs.

* **Hoàn thành dự án triển khai web server toàn diện:**
  * Kết nối đến EC2 instance qua SSH sử dụng key pairs.
  * Cài đặt và cấu hình Apache web server trên Amazon Linux 2023.
  * Truyền tải nhiều file (HTML, CSS, README) sử dụng SCP.
  * Triển khai website hoàn chỉnh với quyền file phù hợp.
  * Cấu hình security groups cho HTTP/HTTPS access.
  * Xác minh khả năng truy cập công khai và chức năng qua browser.
  * Tích lũy kinh nghiệm về quản lý chi phí và tối ưu hóa.
  * Phát triển kỹ năng troubleshooting cho các vấn đề triển khai thường gặp.

* **Phát triển kỹ năng AWS cơ bản** bao gồm quản lý tài khoản, các khái niệm networking cơ bản, và cấp phát tài nguyên cloud.
